To insert a interactive R cell in the content.html file of a page, you can use the following syntax:

<r-cell># This is an R cell
print("Hello, Atomic Learning!")</r-cell>

This will render as the following interactive R cell:

# This is an R cell print("Hello, Atomic Learning!")

When defining a R cell, it is advised to begin the R code immediately after the opening <r-cell> tag and to include the closing </r-cell> tag immediately after the end of the code. It is also advised not to indent code by any spaces or tabs, unless this is required for the syntax of the R code itself (e.g. within functions or loops). Finally, it is recommended to use the closing </r-cell> tag on a new line immediately after the last line of R code. These practices help ensure that the R code in the cell does not have any unnecessary leading or trailing blank lines or spaces.
